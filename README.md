# Monty Hall Problem Simulation
This repository contains a tiny [Python script][PY] to simulate multiple
games of the [Monty Hall problem][MHP] and observe that the success rate
for players who switch doors during the game have a 2/3 chance of
winning the car whereas players who stick to their initial choice have
only a 1/3 chance.

[PY]: ./monty.py
[MHP]: https://en.wikipedia.org/wiki/Monty_Hall_problem
